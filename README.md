# Here is my labs

In folder I, labs from sixth to twelfth are written in FreePascal, so they only have packages compiled for Linux. The rest are written in PascalABC.Net, and they can run on both Windows and Linux

### For Linux
You can also run this binaries on your GNU/Linux OS. You should install mono for your system. For an example, on Ubuntu, this package called `mono-complete`, on ArchLinux - `mono`.
