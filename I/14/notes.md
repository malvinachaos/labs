## МЕТОД ДЕЛЕНИЯ ПОПОЛАМ. 
Применяется для нахождения корня ур-ия f(x) = 0 на отрезке [a,b] для 
непрерывной функции. Работает при условии, что на концах отрезка, 
содержащего корень, функция должна иметь разные знаки. 

Суть метода заключается в последов-м _стягивании_ отрезка к корню:
_let_ _l_ - левая граница, а _r_ - правая граница отрезка. Изначально,
_l = a_ и _r = b_. [l, r] делится пополам точкой _c_ и выбирается та
половина отрезка, на концах которой функция имеет разные знаки: если это
левая половина, _r = c_, иначе _l = c_. Процесс деления идёт до тех пор,
пока не выполнится условие
                            |r - l| ⩽ ε и |f(x)| ⩽ ε. 
```
.------------------------------------------------------------------------.
|             ↑ y                                                        |
|             │                                                          |
|             │                                          /               |
|             │                                        _/                |
|             │                                       / ┆                |
|             │                                      /  ┆                |
|             │                                   __/   ┆                |
|             │                             __----      ┆                |
|             │          a         c       /            ┆                |
|─────────────┼──────────┆─────────┆───────|────────────┆───────────────→|
|             │          ┆         ┆___--  x            b               x|
|             │          ┆_____----┆                                     |
|             │ _____----┆                                               |
|             │                                                          |
|             │                                                          |
*------------------------------------------------------------------------*
```


## МЕТОД ПРОСТЫХ ИТЕРАЦИЙ.
Используется в том случае, если уравнение можно выразить в виде x = f(x).
Тогда корень уравнения является пересечением прямой y=x и кривой y=f(x).
Данный метод записывается в виде рекуррентной формулы:
```
        x_0 = x начальное
        x_i = f(x_{i-1}) для i ∈ ℕ.
```
образующая последовательность _x\_1, x\_2, x\_3..._, сходящуюся к корню
уравнения при определённых условиях. Повторяем процесс до выполнения ус-я
```
            |x_i - x_{i-1}| ⩽ ε
```
```
.------------------------------------------------------------------------.
|             ↑ y                                                        |
|             │      .-__  *    x = y                                    |
|             │     /  ↑→*                                               |
|             │   _/→→→*  \                                              |
|           __│__/ ↑ * |                                                 |
|             │    *   |                                                 |
|             │  * |   |                                                 |
|             │*   |   |                                                 |
|─────────────┼────|───|────────────────────────────────────────────────→|
|           * │    x0  x1                                               x|
|         *   │                                                          |
|       *     │                                                          |
|     *       │                                                          |
*------------------------------------------------------------------------*
```
